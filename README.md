## MotionLayoutSample

MotionLayoutの使い方について説明するためのサンプルプロジェクトです。

### Sample1Activity

transitionToStateを使った状態遷移を解説します。

### Sample2Activity

OnSwipeを使った、Viewへの操作をトリガーにした状態遷移を解説します。

### Sample3Activity

RecyclerViewのスクロールをトリガーとした状態遷移を解説します。

### Sample4Activity

OnClickを使った、Viewへのクリックをトリガーにした状態遷移を解説します。
